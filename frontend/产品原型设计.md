# Multipass 远程管理平台 - 产品原型设计文档

## 1. 产品概述

Multipass 远程管理平台是一个基于 Web 的虚拟机管理系统，提供用户中心、虚拟机管理、Web 远程管理、目录挂载、OpenClaw 管理和远程访问管理六大核心模块。

---

## 2. 用户中心模块

### 2.1 用户注册
- **页面元素**：
  - 用户名输入框（必填，3-20 字符）
  - 邮箱输入框（必填，格式验证）
  - 密码输入框（必填，8-20 字符，含大小写字母和数字）
  - 确认密码输入框（必填）
  - 注册按钮
  - 已有账号？登录 链接

### 2.2 用户登录
- **页面元素**：
  - 用户名/邮箱输入框
  - 密码输入框
  - 记住我 复选框
  - 登录按钮
  - 忘记密码？链接
  - 没有账号？注册 链接

### 2.3 个人信息
- **页面元素**：
  - 头像上传区域
  - 用户名（只读）
  - 邮箱（可编辑，需验证）
  - 昵称（可编辑）
  - 手机号（可编辑，可选）
  - 创建时间（只读）
  - 最后登录时间（只读）
  - 保存按钮

### 2.4 安全设置
- **页面元素**：
  - 修改密码表单
    - 当前密码
    - 新密码
    - 确认新密码
  - 两步验证开关
  - 登录日志列表（时间、IP、地点、设备）
  - 活跃会话管理（设备、地点、登录时间、操作 - 下线）

---

## 3. 虚拟机管理模块

### 3.1 虚拟机列表
- **页面元素**：
  - 搜索框（按名称/IP 搜索）
  - 筛选器（状态：运行中/已停止/已暂停）
  - 新建虚拟机 按钮
  - 虚拟机卡片/表格
    - 名称
    - 状态标签（运行中/已停止/已暂停）
    - IP 地址
    - CPU/内存/磁盘使用率
    - 操作按钮（启动/停止/重启/详情/删除）

### 3.2 创建虚拟机
- **页面元素**：
  - 虚拟机名称输入框
  - 镜像选择下拉框（Ubuntu/CentOS/Debian 等）
  - CPU 核心数选择器（1-8 核）
  - 内存大小选择器（1GB-16GB）
  - 磁盘大小选择器（10GB-500GB）
  - 网络配置（桥接/NAT）
  - 高级设置（折叠面板）
    - SSH 密钥上传
    - 初始化脚本
  - 创建按钮
  - 取消按钮

### 3.3 虚拟机操作
- **快捷操作菜单**：
  - 启动
  - 停止
  - 重启
  - 暂停/恢复
  - 创建快照
  - 删除

### 3.4 虚拟机详情
- **页面元素**：
  - 基本信息卡片
    - 名称、状态、IP
    - 镜像、CPU、内存、磁盘
    - 创建时间、更新时间
  - 资源监控图表
    - CPU 使用率（实时曲线）
    - 内存使用率（实时曲线）
    - 磁盘 IO（实时曲线）
    - 网络流量（实时曲线）
  - 快照列表
    - 快照名称、创建时间、大小
    - 操作（恢复/删除）
  - 操作日志
    - 时间、操作类型、操作结果

---

## 4. Web 远程管理模块

### 4.1 WebShell
- **页面元素**：
  - 终端模拟器区域（xterm.js）
  - 连接状态指示器
  - 工具栏
    - 断开连接
    - 清屏
    - 全屏
    - 字体大小调整
    - 主题切换（亮/暗）
  - 快捷命令面板（可选）

### 4.2 远程管控
- **页面元素**：
  - 文件管理器
    - 目录树
    - 文件列表（名称、大小、修改时间）
    - 操作按钮（上传/下载/新建/删除/重命名）
  - 进程管理
    - 进程列表（PID、名称、CPU、内存）
    - 操作（结束进程）
  - 服务管理
    - 服务列表（名称、状态）
    - 操作（启动/停止/重启）

### 4.3 日志查看
- **页面元素**：
  - 日志类型选择器（系统日志/应用日志/安全日志）
  - 时间范围选择器
  - 关键词搜索框
  - 日志级别筛选（INFO/WARN/ERROR）
  - 日志内容区域（支持实时刷新）
  - 导出按钮（下载日志文件）

## 5. 目录挂载模块

### 5.1 挂载管理
- **页面元素**：
  - 挂载列表
    - 挂载名称
    - 宿主机路径
    - 虚拟机路径
    - 状态（已挂载/未挂载）
    - 操作（挂载/卸载/编辑/删除）
  - 新建挂载 按钮
  - 新建挂载表单
    - 挂载名称
    - 宿主机路径选择器
    - 虚拟机路径输入框
    - 读写权限选择（只读/读写）
    - 自动挂载开关

### 5.2 OpenClaw 专属配置
- **页面元素**：
  - 工作空间映射列表
    - OpenClaw 工作空间路径
    - 虚拟机映射路径
    - 同步状态
  - 技能目录配置

    - 自动同步开关
  - 配置文件同步
    - openclaw.json 映射
    - 工具配置文件映射

## 6. OpenClaw 管理模块

### 6.2 管控配置
- **页面元素**：

    - 默认模型选择

    - 模型切换策略
  - 工具配置
    - 已启用工具列表
    - 工具权限管理
  - 技能配置
    - 已安装技能列表
    - 技能启用/禁用

### 6.3 监控面板
- **页面元素**：
  - 资源使用概览
    - CPU 使用率
    - 内存使用率
    - 磁盘使用率
  - 请求统计
    - 今日请求数
    - 平均响应时间
    - 错误率
  - 会话监控
    - 活跃会话数
    - 会话列表

### 6.4 工作空间管理
- **页面元素**：
  - 工作空间列表
    - 名称、路径、大小
    - 文件数量
    - 操作（进入/备份/删除）

    - 总大小

## 9. 技术栈

- **前端框架**: Vue 3.4+
- **构建工具**: Vite 5+
- **UI 组件库**: Element Plus 2.5+
- **状态管理**: Pinia 2.1+
- **路由管理**: Vue Router 4.3+
- **HTTP 客户端**: Axios 1.6+
- **终端组件**: xterm.js 5.3+
- **图表库**: ECharts 5.4+
- **代码规范**: ESLint + Prettier

## 10. 设计规范

### 10.1 色彩方案
- **主色调**: #409EFF (Element Plus 默认蓝)
- **成功色**: #67C23A
- **警告色**: #E6A23C
- **危险色**: #F56C6C
- **信息色**: #909399

### 10.2 布局规范
- **顶部导航栏高度**: 60px
- **侧边栏宽度**: 220px（可折叠）
- **内容区域边距**: 20px
- **卡片圆角**: 4px

### 10.3 响应式断点
- **移动端**: < 768px
- **平板端**: 768px - 1024px
- **桌面端**: > 1024px
