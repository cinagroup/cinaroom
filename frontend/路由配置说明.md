## 路由守卫

```javascript
router.beforeEach((to, from, next) => {
  // 设置页面标题
  document.title = to.meta.title 
    ? `${to.meta.title} - Multipass 远程管理平台` 
    : 'Multipass 远程管理平台'
  
  const token = localStorage.getItem('token')
  
  // 公开路由直接放行
  if (to.meta.requiresAuth === false) {
    next()
  } else {
    // 受保护路由检查 Token
    if (token) {
      next()
    } else {
      next('/login') // 重定向到登录页
    }
  }
})
```

## 路由元信息

每个路由都包含 `meta` 对象：

```javascript
{
  title: '页面标题',      // 用于浏览器标题和面包屑
  icon: 'IconName',     // Element Plus 图标名
  requiresAuth: false   // 是否需要登录（默认 true）
}
```

## 动态路由参数

以下路由使用动态参数：

- `/vms/:id` - 虚拟机详情
- `/shell/:id` - WebShell
- `/remote/:id` - 远程管控
- `/logs/:id` - 日志查看

可通过 `route.params.id` 获取虚拟机 ID。
