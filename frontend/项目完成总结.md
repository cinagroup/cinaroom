## 使用说明

### 安装依赖
```bash
cd /root/.openclaw/workspace/multipass-frontend
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000 查看应用

### 构建生产版本
```bash
npm run build
```
