# 1.16.1

A patch release fixing issues with certificates and external directories, as well as in mounts:


## New features & improvements

- [cinaseekd] regenerate gRPC server certificate on root ca mismatch ([#4218](https://github.com/canonical/cinaseek/pull/4218))
- Certificates inaccessible when MULTIPASS_STORAGE is set ([#4246](https://github.com/canonical/cinaseek/pull/4246))
- [sshfs-mount] make `chown` return 0 on Windows ([#4288](https://github.com/canonical/cinaseek/pull/4288))

See also the [v1.16.1](https://github.com/canonical/cinaseek/milestone/34?closed=1) milestone and the full diff: https://github.com/canonical/cinaseek/compare/v1.16.0...v1.16.1
