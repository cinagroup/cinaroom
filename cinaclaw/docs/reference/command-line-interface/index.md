# Command-line interface

> See also: [Instance](/explanation/instance), [Service](/explanation/service)

The **`cinaclaw`** CLI (command line interface) client is used to communicate with the cinaclaw service to create, manage, and interact with cinaclaw instances using various sub-commands.

You can use `cinaclaw help <command>` to display more information on the purpose and available options of each command.
